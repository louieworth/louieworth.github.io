Python args, *kwargs 的用法和用处#

Why we need these two in __init__

当一个类需要继承另外一个类的时候，例如Myfoo需要继承Foo，假设我们不知道Foo有哪些参数，而只是关于Foo的一些扩展而己，（另一种情况是将来Foo可能会有变化），这时你就可以利用args, *kwargs 来代表Foo的所有参数，传入到MyFoo
