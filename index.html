Academic Style

No phrasal verbs: Go up/look up

No personal pronouns: I, we, opinion

No contractions: It’ll: It will

No Questions: so why did the project fall?

No personal adverbs: Surprisingly

No vague language: And so on/ etc.

No basic language: like: For example/thing: factor/ lots of: significant amount/amazing: Important
