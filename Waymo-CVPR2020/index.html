Now. there’s more interesting research which is based on an end-to-end branch. However, this end-to-end branch can not match the production of the self-driving system, which is led by the leading companies.

Why difficult?#

Modular#

  • Hierarchical planning: Router, Planer, Controls

  • Early part. In Perception part, we can use supervised learning and self-supervised learning.

  • When it comes Planer, actions affect the environment, supervised learning is not enough. We need to take care about cost function learning, imitation learning, reinforce learning.

A Modular autonomous driving system also end to end?
